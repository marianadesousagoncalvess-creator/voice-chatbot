🎤 Captura de Voz em Python

Este projeto foi desenvolvido como etapa inicial de um assistente virtual por voz.

O objetivo desta fase é capturar o áudio do usuário através do microfone e salvar em um arquivo `.wav`.

🚀 Funcionalidades

- Gravação de áudio via microfone
- Salvamento do áudio em arquivo `.wav`
- Execução local em Python

🛠️ Tecnologias utilizadas

- Python
- sounddevice
- scipy
